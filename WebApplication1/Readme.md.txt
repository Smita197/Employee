# Employee Management System

The Employee Management System is an ASP.NET Core MVC application that provides a simple interface for managing different types of employees, tracking their vacation days, and performing work-related activities.

## Features

- Three types of employees: Hourly employees, Salaried employees, and Managers.
- Automatic calculation of vacation days for each employee type.
- Work() method to simulate days worked and update vacation days accordingly.
- TakeVacation() method to allow employees to take vacation days.
- Web interface for monitoring and interacting with employee data.

## Technologies Used

- ASP.NET Core MVC
- C#
- HTML
- CSS

## Getting Started

To run the application locally, follow the instructions in the [Getting Started](#getting-started) section of the README file.

## Usage

Visit the web interface at `http://localhost:5000` eg:https://localhost:7262/ (default) to view and interact with employee data. The UI provides controls for working and taking vacations, with real-time updates to reflect changes in employee status.
